human-sensors

This was part of a project for a "dream study". The device was battery powered so the patient could stay at home in his normal environment. The data sampled could be written on sd-card or transfered via bluetooth.

Sensors included:

  • 3-axis leg sensors
  • ecg
  • 8-channel eeg
  • emg
  • eog
  • 3-axis accelerometer for general movement

Hardware setup:

  • <a href="https://www.olimex.com/Products/Duino/STM32/OLIMEXINO-STM32/open-source-hardware" target="_blank">Olimex Olimexino-STM32</a>
  • Olimex Olimexino-STM32 board with STM32F103RBT6 microcontroller
  • Olimex MOD-MAG 3-axis magnetometer MAG3110
  • Olimex MOD-LCD3310 Nokia 3310 LCD
  • Olimex MOD-BT bluetooth module
  • Olimex SHIELD-EKG-EMG ecg/emg shield for STM32 board
  • ti MSP430G2553
  • Maxim DS1337 real-time clock
  • a bunch of standard electronic parts

Software:

  • libmaple
  • library for DS1337
  • library for LCD3310
  • library for MAG3110
  • Ethernet and SdFat library
  • library for MPU6050
